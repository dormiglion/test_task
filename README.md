# Тестовое задание: инвентарь и предметы

## Описание

Проект представляет собой серверную модель системы инвентаря и предметов, реализованную на **Node.js + TypeScript**.

Игроки могут:

* получать предметы;
* выбрасывать предметы на землю;
* подбирать предметы, находящиеся рядом;
* использовать предметы с различным поведением.

Мир работает по системе **тиков (tick)**: предметы, лежащие на земле, существуют ограниченное количество тиков и автоматически удаляются после истечения времени жизни.

## Архитектура

Проект разделён на несколько основных компонентов:

* **GameWorld** — ядро системы, реализующее игровой API и взаимодействие игроков с миром.
* **Player** — хранение состояния игрока и логика работы с инвентарём.
* **Items** — иерархия предметов, наследующихся от `BaseItem`.
* **InventoryStorage** — асинхронный слой хранения данных, имитирующий работу базы данных.
* **ItemFactory** — фабрика создания экземпляров предметов по сохранённому состоянию.

## Реализованные возможности

* асинхронное хранение инвентарей;
* конфигурируемые параметры игры;
* выдача предметов;
* выбрасывание и подбор предметов;
* проверка расстояния до предмета при подборе;
* поддержка стакающихся предметов;
* экипировка оружия и брони;
* использование предметов;
* автоматическое удаление предметов с земли по истечении времени жизни;
* защита от конкурентного подбора одного предмета несколькими игроками.

## Конфигурация

Основные игровые параметры вынесены в конфигурацию:

* радиус подбора предметов;
* время жизни предметов на земле;
* максимальное здоровье и броня;
* границы карты;

## Запуск проекта

Установить зависимости:

```bash
npm install
```

Запустить проект:

```bash
npm start
```
